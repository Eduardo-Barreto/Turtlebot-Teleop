# Turtlebot-Teleop
Um simples teleop do turtlebot3. Desenvolvido como atividade ponderada no inteli

Para rodar, vá para `src`.

Em `src/teleop_ws` basta rodar
```sh 
source run.sh
```

Em outro terminal, vá para `src/teleop_keys` rode
```sh 
python3 main.py
```
